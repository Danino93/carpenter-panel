# 🚀 מדריך מהיר לבניית APK

## ✅ מה כבר מוכן

1. ✅ Capacitor מותקן
2. ✅ פלטפורמת אנדרואיד נוספה
3. ✅ האפליקציה נבנתה (`dist/`)

## 📋 השלבים הבאים

### 1. התקן Android Studio
- הורד מ-[developer.android.com/studio](https://developer.android.com/studio)
- התקן (זה לוקח זמן - ~2GB)
- פתח Android Studio פעם אחת להגדרה ראשונית

### 2. פתח את הפרויקט ב-Android Studio

```bash
npx cap open android
```

זה יפתח את Android Studio עם הפרויקט.

### 3. בנה APK

ב-Android Studio:
1. לחץ על **Build** → **Build Bundle(s) / APK(s)** → **Build APK(s)**
2. חכה לסיום (2-5 דקות)
3. כשיסיים, לחץ על **locate** או לך ל:
   ```
   android/app/build/outputs/apk/debug/app-debug.apk
   ```

### 4. התקן על הטלפון

**דרך 1: USB (קל)**
1. חבר את הטלפון למחשב
2. הפעל **USB Debugging** בטלפון:
   - הגדרות → אודות הטלפון → לחץ 7 פעמים על "מספר הבנייה"
   - חזור → אפשרויות מפתח → הפעל "דיבוג USB"
3. ב-Android Studio: לחץ על **Run** (▶️) או `Shift+F10`
4. בחר את הטלפון שלך
5. האפליקציה תתקין אוטומטית!

**דרך 2: העברה ידנית**
1. העתק את `app-debug.apk` לטלפון
2. פתח את הקובץ בטלפון
3. אם צריך: הגדרות → אבטחה → אפשר "מקורות לא ידועים"
4. התקן

## 🎯 מה זה נותן לך?

✅ **APK אמיתי** - אפליקציה עצמאית  
✅ **לא דורש דפדפן** - רץ ישירות על הטלפון  
✅ **נראה מקצועי** - כמו אפליקציה רגילה  
✅ **מוכן ל-Google Play** - אחרי חתימה  

## ⚠️ הערות חשובות

1. **APK Debug** - זה לבדיקות. לייצור צריך APK חתום (ראה `APK_BUILD.md`)
2. **גודל** - ה-APK יהיה ~5-10MB
3. **ביצועים** - האפליקציה תרוץ חלק על הטלפון

## 🔄 עדכונים

כשאתה משנה קוד:
1. `npm run build` - בנייה מחדש
2. `npx cap sync` - סנכרון עם Android
3. ב-Android Studio: **Build** → **Rebuild Project**

## 📱 מה הלאה?

אחרי שיש לך APK שעובד:
- אפשר לפרסם ב-Google Play (צריך APK חתום)
- אפשר לחלוק עם אחרים (שלח את ה-APK)
- אפשר להתקין על כמה טלפונים

---

**הערה:** אם יש בעיות, בדוק ש-Android Studio מותקן ונפתח לפחות פעם אחת לפני `npx cap open android`.
