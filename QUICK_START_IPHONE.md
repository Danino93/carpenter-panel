# 📱 התקנת האפליקציה על אייפון - הדרך הכי קלה!

## ✅ פתרון מהיר - PWA (לא צריך Mac!)

**PWA = Progressive Web App** - אפליקציה שנראית ועובדת כמו אפליקציה רגילה!

## 🚀 איך להתקין על אייפון:

### שלב 1: פרסם את האפליקציה
העלה את האפליקציה לשרת (Netlify, Vercel, GitHub Pages, וכו')

**או** הרץ מקומית:
```bash
npm run dev
```
ואז פתח ב-Safari באייפון את הכתובת (לדוגמה: `http://192.168.1.100:5173`)

### שלב 2: פתח ב-Safari באייפון
1. פתח את **Safari** (לא Chrome!)
2. לך לכתובת של האפליקציה

### שלב 3: הוסף למסך הבית
1. לחץ על כפתור **"שתף"** (⬆️) בתחתית המסך
2. גלול למטה ולחץ **"הוסף למסך הבית"**
3. תן שם: **"פאנל נגרות"**
4. לחץ **"הוסף"**

### שלב 4: מוכן! 🎉
- האפליקציה תופיע על מסך הבית עם האייקון שיצרנו!
- תראה כמו אפליקציה רגילה
- תעבוד גם בלי אינטרנט (אם יש Service Worker)
- עם הלוגו והאייקון היפים!

## 🎯 יתרונות PWA:
- ✅ **עובד מיד** - לא צריך Mac
- ✅ **לא צריך App Store** - התקנה ישירה
- ✅ **עדכונים אוטומטיים** - רק רענון
- ✅ **עובד גם בלי אינטרנט** - אם יש Service Worker
- ✅ **נראה כמו אפליקציה רגילה** - עם אייקון על מסך הבית

## 📝 הערות:
- צריך לפרסם את האפליקציה ב-**HTTPS** (לא HTTP)
- עובד מצוין על אייפון ו-iPad
- נראה בדיוק כמו אפליקציה רגילה!
