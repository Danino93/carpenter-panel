# 🎨 הוראות יצירת אייקונים לאפליקציה

## אייקונים נדרשים

האפליקציה צריכה 2 אייקונים:
- `icon-192.png` - 192x192 פיקסלים
- `icon-512.png` - 512x512 פיקסלים

## איך ליצור אייקונים

### אפשרות 1: כלי אונליין (קל)
1. היכנס ל-[favicon.io](https://favicon.io) או [realfavicongenerator.net](https://realfavicongenerator.net)
2. העלה תמונה או צור אייקון
3. הורד את הקבצים
4. העתק את `android-chrome-192x192.png` ל-`public/icon-192.png`
5. העתק את `android-chrome-512x512.png` ל-`public/icon-512.png`

### אפשרות 2: יצירה ידנית
1. צור תמונה 512x512 פיקסלים
2. הוסף לוגו/אייקון של נגרות (מסור, פטיש, וכו')
3. שמור כ-PNG
4. צור עותק ב-192x192

### אפשרות 3: אייקון פשוט (זמני)
אפשר להשתמש באייקון פשוט עם טקסט:
- רקע כחול (#3b82f6)
- טקסט: "🔨" או "נגרות"
- גופן גדול

## מיקום הקבצים

הקבצים צריכים להיות ב:
```
public/
  ├── icon-192.png
  ├── icon-512.png
  └── manifest.json
```

## בדיקה

לאחר הוספת האייקונים:
1. הרץ `npm run dev`
2. פתח את DevTools (F12)
3. לך ל-Application → Manifest
4. בדוק שהאייקונים נטענים

## התקנה על הטלפון

לאחר הוספת האייקונים, האפליקציה תראה מקצועית יותר בהתקנה על הטלפון!
