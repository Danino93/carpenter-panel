# פאנל נגרות - מחשבון חומרים מקצועי

פאנל נגרות הוא כלי מקצועי לחישוב חומרים והצעת מחיר לנגרים. המערכת מחשבת אוטומטית רשימת חיתוכים מדויקת, רשימת קניות, ועלויות מפורטות.

## ⭐ תכונות עיקריות

- ✂️ **חישוב חיתוכים מדויק** - רשימת כל החתיכות עם מידות מדויקות
- 🛒 **רשימת קניות** - כמה לוחות סטנדרטיים (244x122) לקנות, איך לחתוך אותם בצורה יעילה
- 💰 **חישוב עלויות** - כל העלויות מפורטות, הצעת מחיר ללקוח
- 👥 **ניהול לקוחות** - רשימת לקוחות, פרטי קשר, היסטוריית עבודות
- 📋 **תבניות עבודות** - עבודות חוזרות לשימוש מהיר
- 📊 **דוחות וסטטיסטיקות** - ניתוח עבודות, הכנסות, לקוחות פעילים
- 🔐 **אבטחה** - התחברות מאובטחת עם Supabase
- 📱 **PWA + APK** - עובד על הטלפון כמו אפליקציה רגילה
- ✅ **Validation מלא** - בדיקת מידות הגיוניות, אזהרות

## 🚀 התחלה מהירה

### התקנה

```bash
npm install
npm run dev
```

האפליקציה תיפתח ב-`http://localhost:5173`

### בנייה לייצור

```bash
npm run build
```

### בניית APK לאנדרואיד

ראה `QUICK_START_APK.md` או `APK_BUILD.md`

## 🎯 שימוש בסיסי

1. **יצירת עבודה חדשה:**
   - לחץ על "➕ עבודה חדשה"
   - עבר דרך 7 השלבים
   - צפה בתוצאות

2. **רשימת קניות:**
   - לאחר החישוב, גלול ל-"🛒 רשימת קניות"
   - תראה רשימה מפורטת של מה לקנות

3. **ניהול לקוחות:**
   - לחץ על "👥 לקוחות"
   - הוסף/ערוך/מחק לקוחות

4. **תבניות:**
   - שמור עבודות חוזרות כתבניות
   - השתמש בתבנית ליצירת עבודה חדשה

## ⚙️ הגדרות

### עדכון מחירים
