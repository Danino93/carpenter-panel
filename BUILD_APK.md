# 📱 הוראות בניית APK

## ✅ מה כבר מוכן:
- ✅ Capacitor מוגדר
- ✅ פלטפורמת Android נוספה
- ✅ אייקון יפה נוצר (פטיש ומסור עם גרדיאנט ססגוני)
- ✅ האפליקציה נבנתה בהצלחה

## 🚀 שלבים לבניית APK:

### שלב 1: פתיחת Android Studio
```bash
npx cap open android
```

### שלב 2: בניית APK ב-Android Studio

1. **Android Studio יפתח את הפרויקט**
2. **חכה שהסינכרון יסתיים** (Gradle Sync)
3. **לחץ על:** `Build` → `Build Bundle(s) / APK(s)` → `Build APK(s)`
4. **חכה לסיום הבנייה** (כמה דקות)

### שלב 3: מציאת ה-APK

לאחר הבנייה, ה-APK יהיה ב:
```
android/app/build/outputs/apk/debug/app-debug.apk
```

### שלב 4: העברה לטלפון

**אפשרות 1: USB (הכי קל)**
1. חבר את הטלפון למחשב עם כבל USB
2. הפעל **USB Debugging** בטלפון:
   - הגדרות → אודות הטלפון → לחץ 7 פעמים על "מספר הבנייה"
   - חזור → אפשרויות מפתח → הפעל "דיבוג USB"
3. ב-Android Studio: לחץ על **Run** (▶️) או `Shift + F10`
4. בחר את הטלפון שלך
5. האפליקציה תותקן אוטומטית!

**אפשרות 2: העברה ידנית**
1. העתק את הקובץ `app-debug.apk` לטלפון (דרך USB, אימייל, Google Drive, וכו')
2. בטלפון: פתח את הקובץ
3. אם צריך: אפשר התקנה מ-"מקורות לא ידועים"
4. לחץ "התקן"

## 🎨 האייקון שנוצר:

האייקון כולל:
- 🎨 **גרדיאנט ססגוני** (סגול-כחול-ורוד)
- 🔨 **פטיש** - כלי נגרות קלאסי
- 🔪 **מסור** - כלי נוסף
- ✨ **אפקטי זוהר** - נקודות זוהרות אנימציה

## 📝 הערות חשובות:

- **APK Debug** - זה APK לבדיקה, לא לייצור
- **לייצור** - צריך APK חתום (ראה `APK_BUILD.md`)
- **הגדרות** - כל ההגדרות נמצאות ב-`capacitor.config.ts`

## 🆘 בעיות נפוצות:

**"Gradle sync failed"**
- פתח את `android` ב-Android Studio
- לחץ על "Sync Project with Gradle Files"

**"SDK not found"**
- ב-Android Studio: Tools → SDK Manager
- התקן את Android SDK

**"Build failed"**
- בדוק שיש לך Java JDK מותקן
- בדוק את ה-Gradle version
